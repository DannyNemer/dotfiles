	sh install.sh

Setup Dropbox (after installing via `brew`).

Create symbolic links for dotfiles:
	sh link.sh

Restore application preferences:
	mackup restore

Setup Alfred sync.

Sync Desktop with Dropbox.

	sh macos.sh
